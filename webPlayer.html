Collada viewer:

Here is a prototype to display collada files with only javascript libraries.
In the future, this player may become the arboris viewer: it should ease visualization (no compilation, cross-platform, etc.).
  • The animation was generated with Arboris-python simulator to get an animated collada file
  • The 3D graphic library is three.js
  • The user interface is developed with jQuery UI
It is possible to move the scene: (left-click: rotate, right-click: translate, middle-click: zoom )
